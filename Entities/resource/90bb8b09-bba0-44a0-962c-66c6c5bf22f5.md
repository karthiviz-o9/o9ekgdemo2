# Data Dictionary: Resource (Manufacturing & Distribution)

## Overview
The `Resource` entity defines the specific physical assets or human labor pools used to execute tasks across the supply chain. This includes automated factory machinery, distribution center equipment (like forklifts), facility assets (like loading docks), and skilled workers.

## Entity Details
* **Entity Name:** `Resource`
* **Primary Key:** `Resource_ID`
* **Description:** Contains master data for the type, capacity measurement, and financial costing of the assets that perform work in manufacturing plants and warehouses.

---

## Attribute Dictionary

| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Resource_ID** | String(50) | Primary Key, Not Null | Unique identifier for the specific machine, vehicle, dock, or labor group. | `ROBOT-ARM-02`, `FORKLIFT-RT-05`, `DOCK-OUT-12` |
| **Resource_Name** | String(100) | Not Null | Descriptive name of the resource. | `Kuka Welding Robot`, `5000lb Reach Truck`, `Outbound Dock Door 12` |
| **Resource_Type** | String(30) | Not Null | Categorizes the broad nature or function of the resource. | `Machine`, `Human/Labor`, `Material Handling`, `Facility Asset`, `Tooling` |
| **Capacity_UOM** | String(10) | Not Null | The unit of measure used to track the availability, utilization, and consumption of this resource. | `HR` (Hours), `MIN` (Minutes) |
| **Hourly_Cost_Rate**| Decimal(10,4)| Optional | The standard financial cost of running or staffing this resource for one hour (used for operational costing). | `45.5000`, `18.0000` |
