# Data Dictionary: Transactions (Pipeline Inventory)

## Overview
The `Transactions` entity acts as a ledger for all incoming, expected inventory. It tracks materials that are not yet available for use or sale, but are actively in the pipeline, such as Work-in-Progress (WIP) on the factory floor, In-Transit goods on a truck/ship, and On-Order goods waiting to be fulfilled by a vendor.

## Entity Details
* **Entity Name:** `Transactions`
* **Primary Key:** `Transaction_ID`
* **Description:** Contains transactional data used by planning engines to calculate future inventory balances and availability dates.

---

## Attribute Dictionary

| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Transaction_ID** | String(50) | Primary Key, Not Null | Unique identifier for the transaction or ledger entry. | `TXN-884920`, `WIP-1029` |
| **Reference_Order**| String(50) | Not Null | The originating document number (e.g., Purchase Order, Work Order, Transfer Order). | `PO-45992`, `WO-1002` |
| **Item_ID** | String(50) | Foreign Key, Not Null | The specific item being manufactured, purchased, or moved. | `FG-9921`, `RM-10045` |
| **Source** | String(50) | Foreign Key, Optional | The facility, vendor, or work center supplying the materials (null if starting from scratch). | `VEND-001`, `PLANT-004` |
| **Destination** | String(50) | Foreign Key, Not Null | The facility or location where the inventory is expected to arrive or be completed. | `PLANT-004`, `DC-US-WEST` |
| **Inventory_State**| String(30) | Not Null | The current status of the pipeline inventory. | `In-Transit`, `On-Order`, `WIP` |
| **Quantity** | Decimal(10,2) | Not Null | The amount of the item expected to be received or completed. | `500.00`, `1250.50` |
| **UOM** | String(10) | Not Null | The unit of measure for the transaction quantity. | `EA`, `KG`, `PALLET` |
| **Expected_Date** | Date | Not Null | The exact date the inventory is scheduled to arrive at the destination or complete manufacturing. | `2026-04-15` |
