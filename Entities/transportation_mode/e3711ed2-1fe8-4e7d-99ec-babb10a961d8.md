# Data Dictionary: Transportation Mode

## Overview
The `Transportation Mode` entity defines the different methods and vehicles used to physically move goods across the supply chain network. Planners use this data to balance trade-offs between speed, cost, and environmental impact (carbon footprint) when routing materials.

## Entity Details
* **Entity Name:** `Transportation_Mode`
* **Primary Key:** `Transportation_Mode`
* **Description:** Contains master data for logistics methods, including their relative speeds, cost profiles, and sustainability metrics.

---

## Attribute Dictionary

### 1. Core Identification & Classification
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Transportation_Mode** | String(30) | Primary Key, Not Null | Unique identifier for the method of transportation. | `FTL`, `LTL`, `OCEAN`, `AIR`, `RAIL` |
| **Mode_Name** | String(100) | Not Null | The full descriptive name of the transportation mode. | `Full Truckload`, `Ocean Freight`, `Air Express` |
| **Equipment_Type** | String(50) | Optional | The specific type of equipment or container typically used. | `53ft Trailer`, `40ft Container`, `Reefer` |

### 2. Sustainability & Environmental
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **CO2_Emission_Factor**| Decimal(10,4) | Not Null | Standardized metric representing the carbon footprint (e.g., grams of CO2 emitted per ton-mile or ton-kilometer). | `62.0000` (Truck), `8.0000` (Ocean) |
| **Sustainability_Rating**| String(10) | Optional | An internal or external grading of the mode's environmental impact. | `A` (Green/Rail), `D` (High Impact/Air) |
| **Energy_Source** | String(30) | Optional | The primary fuel or energy type powering this mode. | `Diesel`, `Electric`, `Bunker Fuel` |

### 3. Planning Constraints & Performance
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Speed_Category** | String(20) | Not Null | Categorical representation of the mode's transit speed. | `Express`, `Standard`, `Economy` |
| **Relative_Cost_Tier** | String(20) | Not Null | General cost classification used for high-level planning trade-offs. | `Premium`, `Moderate`, `Low` |
| **Reliability_Score** | Decimal(5,2) | Optional | Historical on-time percentage or reliability index for this mode (0-100%). | `98.50`, `85.00` |
