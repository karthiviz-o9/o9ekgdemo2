# Data Dictionary: Manufacturing Routing

## Overview
The `Manufacturing Routing` entity defines the order of operations required to manufacture, assemble, or process an item on the factory floor. It links an item to the specific machines, labor pools, and time required to produce it.

## Entity Details
* **Entity Name:** `Manufacturing_Routing`
* **Primary Key:** `Routing_ID`
* **Description:** Contains the operational steps, resources (machines/humans), and standard times used by capacity planning and shop floor control systems.

---

## Attribute Dictionary

| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Routing_ID** | String(50) | Primary Key, Not Null | Unique identifier for the specific routing step. | `RTG-9921-010` |
| **Item_ID** | String(50) | Foreign Key, Not Null | The manufactured part or sub-assembly this routing applies to (links to `SupplyChain_Item`). | `FG-9921` |
| **Work_Center** | String(50) | Foreign Key, Not Null | The specific machine, assembly line, or human labor group performing the work. | `CNC-Machine-01`, `Assembly-Cell-A` |
| **Setup_Time** | Decimal(8,2) | Optional | The standard time required to prepare the machine or work area before production begins. | `45.00`, `120.00` |
| **Run_Time** | Decimal(10,4)| Not Null | The standard time required to process or manufacture a single unit (or base batch) at this operation. | `2.5000`, `0.2500` |
| **Time_UOM** | String(10) | Not Null | The unit of measure for both setup and run times. | `MIN` (Minutes), `HR` (Hours) |
