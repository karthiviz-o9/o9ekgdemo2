# Data Dictionary: Product (Commercial & Marketing)

## Overview
The `Product` entity represents a good or service as it is marketed, sold, and presented to the customer. It focuses on features, commercial variants, market positioning, and lifecycle rather than operational supply chain metrics.

## Entity Details
* **Entity Name:** `Product`
* **Primary Key:** `Product_ID`
* **Description:** Contains master data related to marketing, product management, sales configurations, and market characteristics.

---

## Attribute Dictionary

### 1. Core Identity & Branding
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Product_ID** | String(50) | Primary Key, Not Null | Unique identifier for the commercial product (often a base model or family code). | `PRD-SMART-X1` |
| **Product_Name** | String(150) | Not Null | The customer-facing, marketable name of the product. | `Galaxy Orion Pro`, `CloudSync Enterprise` |
| **Brand** | String(100) | Not Null | The brand or sub-brand under which the product is sold. | `Acme Corp`, `EcoEssentials` |
| **Product_Line** | String(100) | Optional | The broader portfolio or collection this product belongs to. | `Winter 2026 Collection`, `Professional Series` |

### 2. Features & Variations
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Key_Features** | JSON / Text | Optional | A structured list or block of text highlighting the main selling points and capabilities. | `{"camera": "48MP", "battery": "24hr"}` |
| **Available_Variants** | Array[String] | Optional | List of variant dimensions available for this product (e.g., color, size, material). | `["Color", "Storage Capacity", "Size"]` |
| **Configurable_Options** | Boolean | Not Null | Indicates if the customer can customize features before purchase (e.g., software add-ons, car trims). | `TRUE`, `FALSE` |
| **Warranty_Type** | String(50) | Optional | The standard commercial warranty offered to the customer. | `1-Year Limited`, `Lifetime` |

### 3. Market Characteristics & Pricing
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Target_Demographic** | String(255) | Optional | Description or code representing the primary ideal customer profile (ICP) or age/interest group. | `Gen-Z Gamers`, `B2B Enterprise IT` |
| **MSRP** | Decimal(10,2) | Optional | Manufacturer's Suggested Retail Price (the baseline commercial price). | `999.00`, `149.50` |
| **Pricing_Strategy** | String(50) | Optional | The commercial strategy governing how this product is priced. | `Premium`, `Penetration`, `Freemium` |
| **Est_Market_Share_Pct** | Decimal(5,2) | Optional | The estimated current market share percentage for this product within its specific category. | `15.50`, `4.00` |
| **Primary_Competitors** | Array[String] | Optional | List of direct competitor products in the market. | `["Competitor X20", "Brand Y Elite"]` |
| **Positioning_Statement** | Text | Optional | Internal marketing statement defining how this product is uniquely positioned against competitors. | `"The only eco-friendly option with..."` |

### 4. Commercial Lifecycle
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Marketing_Status** | String(30) | Not Null | The commercial lifecycle phase of the product. | `Pre-Launch`, `Growth`, `Mature`, `Sunset` |
| **Launch_Date** | Date | Optional | The official date the product is/was made available to the public for purchase. | `2026-05-15` |
| **End_Of_Sales_Date** | Date | Optional | The date the product will no longer be commercially available for new orders. | `2028-12-31` |
