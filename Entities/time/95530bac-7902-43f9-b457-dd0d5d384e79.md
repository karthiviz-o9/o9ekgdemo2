# Data Dictionary: Time (Enterprise & Planning Calendar)

## Overview
The `Time` entity (often called the Time Dimension or Date Dimension) provides a standardized calendar for the entire enterprise. It allows operational data (like daily shipments) to be perfectly aligned with planning buckets (like forecasting weeks or financial quarters) to ensure consistency across supply chain and finance teams.

## Entity Details
* **Entity Name:** `Time`
* **Primary Key:** `Date`
* **Description:** Contains granular, pre-calculated calendar and fiscal attributes for every single day over a multi-year horizon (e.g., 10 years past, 10 years future).

---

## Attribute Dictionary

### 1. Core Date Identification
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Date** | Date | Primary Key, Not Null | The standard calendar date used as the primary identifier. | `2026-03-22` |
| **Day_Of_Month** | Integer | Not Null | The numerical day of the month (1-31). | `22` |
| **Day_Of_Week** | Integer | Not Null | The numerical day of the week (e.g., 1=Monday, 7=Sunday). | `7` |
| **Day_Name** | String(10) | Not Null | The text name of the day. | `Sunday`, `Monday` |

### 2. Weekly Planning Attributes (Critical for Supply Chain)
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Week_Of_Year** | Integer | Not Null | The standard week number of the calendar year (1-52/53), typically ISO 8601 standard. | `12`, `45` |
| **Planning_Week** | String(10) | Not Null | Formatted string combining year and week for weekly planning buckets. | `2026-W12` |
| **Week_Start_Date** | Date | Not Null | The date of the first day of this specific planning week (usually Monday or Sunday). | `2026-03-16` |
| **Week_End_Date** | Date | Not Null | The date of the last day of this specific planning week. | `2026-03-22` |

### 3. Monthly & Quarterly Aggregation
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Month_Number** | Integer | Not Null | The numerical month of the year (1-12). | `3`, `11` |
| **Month_Name** | String(15) | Not Null | The full text name of the month. | `March`, `November` |
| **Planning_Month** | String(10) | Not Null | Formatted string for monthly planning buckets. | `2026-M03` |
| **Calendar_Quarter** | Integer | Not Null | The standard calendar quarter (1-4). | `1`, `4` |
| **Planning_Quarter** | String(10) | Not Null | Formatted string for quarterly S&OP (Sales and Operations Planning) rollups. | `2026-Q1` |

### 4. Yearly & Operational Flags
| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Calendar_Year** | Integer | Not Null | The standard 4-digit calendar year. | `2026` |
| **Fiscal_Year** | Integer | Not Null | The financial year this date falls into (which may differ from the calendar year, e.g., starting in April or July). | `2027` |
| **Is_Working_Day** | Boolean | Not Null | Flag indicating if this date is a standard operational working day. | `TRUE`, `FALSE` |
| **Is_Holiday** | Boolean | Not Null | Flag indicating if this date is a recognized corporate or national holiday. | `TRUE`, `FALSE` |
| **Holiday_Name** | String(50) | Optional | The name of the holiday, if applicable. | `New Year's Day`, `Diwali` |
