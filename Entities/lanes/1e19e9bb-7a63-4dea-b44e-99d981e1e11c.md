# Data Dictionary: Distribution Lane

## Overview
The `Distribution Lane` entity defines the simplified physical pathways connecting nodes across the supply chain network.

## Entity Details
* **Entity Name:** `Lanes`
* **Primary Key:** `Lane`
* **Description:** Contains foundational routing master data indicating where materials flow from and to.

---

## Attribute Dictionary

| Attribute Name | Data Type | Constraint | Description | Example Values |
| :--- | :--- | :--- | :--- | :--- |
| **Lane** | String(50) | Primary Key, Not Null | Unique identifier for the specific sourcing lane. | `LANE-10944`, `BOD-US-01` |
| **Lane_Type** | String(30) | Not Null | The classification or category of the distribution movement. | `Internal Transfer`, `Supplier to Plant`, `DC to Customer` |
| **Source** | String(50) | Foreign Key, Not Null | The facility or node supplying the materials. | `PLANT-004`, `VEND-992` |
| **Destination** | String(50) | Foreign Key, Not Null | The facility or node receiving the materials. | `DC-US-WEST`, `STORE-102` |
